### 基础

网页由 3 个部分构成，我们将其称为`三剑客`。  
它们是前端开发者的基础知识、立命之本的技能！  
强烈建议把它们学熟之后，再学后边的知识！

<Mermaid
  chart={`graph TB;
    前端-->Html;
    前端-->Css;
    前端-->JavaScript;
`}
/>

### 插件

有些很好用或常用的功能，我们不希望每次都要重新写。   
于是就单独拎出来，用的时候再引入进来直接使用即可！   
单独拎出来 并放到网上供大家下载使用的东西 就是库。

### 辅助工具

很长一段时间大家都用 css、和 js 来编写页面的样式和交互部分。  
但是由于这两门语言 本身不够优秀，便有了:    
1. 微软发明的JavaScript 的辅助工具 Typescript。  
2. 勤劳的开源者发明的 Css 的辅助工具 Less、Scss。

### 框架

在15 年之前，大家都用三剑客来写网页！  
时间久了，前端开发者们觉得：原生的写法不友好，效率低！  
针对这些问题，于是就有大佬 或 公司发明了一些好用的`框架`。

<Mermaid
  chart={`graph TB;
    框架-->angular;
    框架-->react;
    框架-->vue;
`}
/>

### 构建工具

框架编写的内容 或者 特别新的语法，浏览器都不能直接识别使用。       
需要让`构建工具`编译成浏览器目前能识别内容供其使用。 
> 浏览器只认识 三剑客，即html、css、javaScript 而且不能特别新

<Mermaid
  chart={`graph TB;
    构建工具-->babel
    构建工具-->webpack;
    构建工具-->rollup;
    构建工具-->esbuild;
    构建工具-->vite;
`}
/>

如上就涵盖了 90%以上的前端知识点，学会他，就可以在 前端界 横着走！
