## 基础

网页由 3 个部分构成，我们将其称为`三剑客`。  
它们是前端开发者的基础、立命之本！  
强烈建议把它们学熟之后，再学后边的知识！

## 插件

有些常用好用功能，我不希望每次都重新写。   
于是单独拎出来，用时候再引入使用即可！   
单拎出并放网上供大家下载使用的东西 就是库。

## 辅助工具

很长一段时间大家都用 css 和 js 来编写页面的样式和交互部分，但由于这两门语言本身不够优秀，于是便有了辅助工具:    
1.微软发明的 Typescript。     
2.开源者发明的 Less、Scss。


## 构建工具

框架编写的内容 或者 特别新的语法，浏览器都不能直接识别使用。需要使用`构建工具`将其编译成浏览器目前能识别内容来使用。 
:::tip
浏览器只认识 三剑客，即html、css、javaScript 而且不能特别新!
:::

于是就有人发明了一些好用的`框架`。
