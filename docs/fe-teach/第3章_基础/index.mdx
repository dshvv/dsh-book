网页由三部分构成，分别是html、css、js。

其中html+css来构成网页的布局，而js是沟通用户和浏览器交互的桥梁。

### 布局 ( html+css )
编写网页，就像你绘画一样，   
其中html就是你用画笔素描基本的轮廓。   
而css就是你给素描上色修饰填充。
