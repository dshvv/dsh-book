网页由三部分构成，分别是 html、css、js。

其中 html+css 来构成网页的布局，而 js 是沟通用户和浏览器交互的桥梁。

## 布局 ( html+css )

编写网页，就像你绘画一样，  
其中 html 就是你用画笔素描基本的轮廓。  
而 css 就是你给素描上色修饰填充。

我们来写一个 html 如下： div 相当于画出的一个盒子，里边的内容是 “你好，世界”

<CodeSandbox file="3-1" />

我们用 css 给盒子添些样式: 将盒内字体颜色设为蓝色，盒子的背景设置成粉红色

<CodeSandbox file="3-2" />

其实，不管是多复杂的页面 都可以利用 html+css 完成整个页面的布局（画出来）！

## 交互 ( javaScript )

截至目前，我们写的页面还只是纯静态的，尚不能和浏览器以及页面进行交互。

而通过 js 我们则可以跟页面实现动态交互。

页面上有一个按钮，点一下，你就能看到交互
