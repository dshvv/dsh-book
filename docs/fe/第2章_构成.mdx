## 基础

网页由 3 个部分构成，我们将其称为`三剑客`。  
它们是前端开发者的基础、立命之本！  
强烈建议把它们学熟之后，再学后边的知识！

## 插件

有些常用好用功能，我不希望每次都重新写。   
于是单独拎出来，用时候再引入使用即可！   
单拎出并放网上供大家下载使用的东西 就是库。

## 辅助

很长一段时间大家都用 css 和 js 来编写页面的样式和交互部分，但由于这两门语言本身不够优秀，于是便有了辅助工具:    
1.微软发明的 Typescript。     
2.开源者发明的 Less、Scss。

时间久了就觉得：原生的写法不友好，效率低！  
于是就有人发明了一些好用的`框架`。

## 工程化

随着前端页面越来越复杂，原生三剑客或者jq一把梭时代已经无法满足我们的需求，这时候众多前端工程化的技术和工具纷纷出现。

|  |   |
| --- | --- |
| 脚手架工具 | vue-cli、create-react-app ... |
| 模块化打包 | webpack、rollup |
| 标准化工具 | eslint、styleLint、prettier |
| 自动化构建 | npm script |
| 自动化测试 | jest |
| 自动化部署 | git hook、lint-staged |


如上就涵盖了 90%以上的前端知识，学会后你就可以在 前端界 横着走
