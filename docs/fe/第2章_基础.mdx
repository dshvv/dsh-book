它们是前端开发者的基础、立命之本, 强烈建议把它们学熟之后，再学后边的知识！

## 布局 ( html+css )

编写网页就像你绘画一样，其中 html 就是你用铅笔素描基本的轮廓，而 css 就是你给素描上色修饰填充。  
![fe-2.1](/img/fe/2.1.jpg)

一个网页，可以看作是一个个小盒子构成。我们利用 html+css 来画出一个个小盒子来组装成完整页面。  
![fe-2.2](/img/fe/2.2.png)

<hr />

--- 举个例子 ![fe-2.3](/img/fe/2.3.webp) ---  
我们来写一个 html 如下： div 相当于画出的一个盒子，里边的内容是 “你好，世界”

```html
<div>你好，世界</div>
```

![fe-2.4](/img/fe/2.4.jpg)

我们用 css 给盒子添些样式： 将盒内字体颜色设为蓝色，盒子的背景设置成粉红色

```html
<div style="color:red; background-color: blue;">你好，世界</div>
```

![fe-2.5](/img/fe/2.5.jpg)

至此，我们已经画了一个好看的盒子了！  
而复杂的页面，都是由这些盒子一个个拼起来的。  
所以目前为止，你已经掌握了如何写页面了，看就是这么简单！  
![fe-2.6](/img/fe/2.6.webp)

## 交互 ( javaScript )

截至目前，我们写的页面还只是纯静态的，尚不能和浏览器以及页面进行交互。  
而通过 js 我们则可以跟页面实现动态交互：页面上有一个按钮，点一下

<button
  onClick={() => {
    alert("你好");
  }}
>
  测试
</button>

```html
<button onclick="test">测试</button>
<script>
  function test() {
    alert("你好");
  }
</script>
```

不错，网页编程的最后一门技能`js`你也会了。  
你现在已经具备了开发一个完整项目的能力了，放在几年前 你就可以找到一份月薪几千的前端开发工作了！
![fe-2.7](/img/fe/2.7.webp)
