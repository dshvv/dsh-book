随着 Web 业务日益复杂化和多元化，前端开发已经由以 WebPage 模式为主转变为以 WebApp 模式为主了。

现在随便找个前端项目，都已经不是过去的拼个页面+搞几个 jQuery 插件就能完成的了。

项目复杂了就会产生许多问题，如何进行高效的多人协作？如何保证项目的可维护性？如何提高项目的开发质量？
...

于是便有了 提升开发体验、效率和质量，提升应用的访问性能等手段（工具和技术）。

这就是前端工程化！
