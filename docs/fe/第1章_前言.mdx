前端 就是写网页的！
:::

啥是网页呢？  
你拿起手机或电脑，使用浏览器打开一个网站或链接，映入你眼帘的就是网页。    
![fe-1.1](/img/fe/1.1.png)
![fe-1.2](/img/fe/1.2.png)

之前我们把这个写网页的岗位称之为`切图仔`，现在我们被尊称为 前端`开发工程师` 甚至`大前端`。 
![fe-1.3](/img/fe/1.3.jpg)
 
前端工程师的工资还是很高的！  
既然想入此行，那就好好学习吧！  
![fe-1.3](/img/fe/1.4.jpg)

建议你按照目录大纲 一步步的学习，成为大神
```
第一步：基础
第二步：插件与辅助
第三步：框架
第四步：工程化
```
