网页由三部分构成，分别如下

### 布局 ( html+css )
编写网页，就像你画画一样，   
其中html就是你用画笔素描基本的轮廓。   
而css就是你用各种颜料来给你的素描上色修饰填充。   


我们来写一个html： div相当于画出的一个盒子，里边的内容是 “你好，世界”

接下来我们使用css来给这个盒子添加一些样式: 将盒子内的字体颜色设置为红色，盒子的背景色设置未粉红色
<CodeSandbox file="3-2"/>

不管是多复杂的页面 都可以利用html+css 完成整个页面的布局（画出来）！

### 交互 ( javaScript )
如上的布局，是纯静态的，你还不能和浏览器 页面进行交互。    
通过 javaScript 我们则可以跟页面实现动态交互。
比如页面上有一个按钮，鼠标点一下，你就能看到交互了
