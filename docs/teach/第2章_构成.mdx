### 基础

一个网页由 3 个部分构成，俗称`前端三剑客`。  
这也是前端开发者最基础的、最本质的、立命之本的技能！  
强烈建议把三剑客学的差不多了的之后，在学习框架等后边的其他知识！

### 插件与ui库

有一些很常用的功能，被抽封装成一个个包，放在网上。  
当你需要使用某些功能的时候，就可以下载下来引入使用。  
相当的省事和方便！

### 辅助工具

很长一段时间大家都用 css、和 js 来编写页面的样式和交互部分。  
但是由于这两门语言 本身不够优秀，基于便有了:    
微软发明的JavaScript 的辅助工具 Typescript。  
一些勤劳的开源者发明的 Css 的辅助工具 Less、Scss。

### 框架

至少 2015 年之前，大家都会使用原生的三剑客来编写网页！  
时间久了，写的多了前端开发者们就会觉得：原生的写法不是特别的友好，效率还低！  
于是就有大佬 或 公司发明了一些好用的`前端框架`来提高了页面编写的玩法。

<Mermaid
  chart={`graph TB;
    框架-->angular;
    框架-->react;
    框架-->vue;
`}
/>

### 构建工具

如上这些框架编写的内容 浏览器不能直接识别和使用，需要让`构建工具`最终编译成浏览器能识别的原生 三剑客 供浏览器使用。  
除了上诉外 旧版的浏览器使用新版的 三剑客语法，也不能被其识别，也需要这些构建工具编译成对应旧版本浏览器代码使用。  
所以构建工具也很重要，以下就是常用主流的构建工具：

<Mermaid

/>

如上就涵盖了 90%以上的前端知识点，学会他，就可以在 前端界 横着走！
