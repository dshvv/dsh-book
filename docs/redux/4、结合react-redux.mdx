因为 redux 和 react 组件的组合使用耦合度太高，为了解耦redux官方额外设计了react-redux ，有了它我们便不再需要使用 store 的 subscribe 自己去订阅状态了。

通过 react-redux 帮我们把redux和react相连接，react项目中使用redux将灵活而高效。
