## 这是啥

redux-toolkit 号称自己是 Redux 官方工具包，用于高效的 Redux 开发。

```shell
yarn add @reduxjs/toolkit
```

## chrome 插件支持

原生的写法不支持浏览器使用插件 debug。redux-toolkit 创建的 store 则可以
![4.1](/img/redux/4.1.png)

```jsx title="/src/store/index.jsx"
import { configureStore } from "@reduxjs/toolkit";

// state
const initialState = {
  msg: "张三",
};

// reducer
const reducer = (state = initialState, action) => {
  switch (action.type) {
    case "SET_MSG":
      return { ...state, ...{ msg: action.payload } };
    default:
      return state;
  }
};

// store
export default configureStore({ reducer });
```
